1.BFC

https://zhuanlan.zhihu.com/p/25321647

BFC：块级格式化上下文，指一个独立的块级渲染区域。只有display为block的元素参与，该区域拥有一套渲染规则来约束块级盒子的布局，且与区域外部无关。

如何创建BFC：

方法1：float的值不是none

方法2：position的值不是static或者relative

方法3：display的值为inline-block、flex或者inline-flex

方法4：overflow: hidden

如下所示，如果不给父盒子设置上述条件，因为子盒子都增加了浮动，所以父盒子不会因为子盒子的高度而被撑开，即父盒子没有高度。如果加上上述条件中的一个，则父子会拥有一个被子盒子高度撑开的高度。

最好的办法是 overflow :hidden

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>BFC</title>
    <style>
        .father {
            /* float: left;
            position: absolute;
            display: inline-block; */
            overflow: hidden;
        }

        .son {
            float: left;
            width: 300px;
            margin-right: 20px;
            background-color: cadetblue;
            height: 300px;
        }
    </style>
</head>

<body>
    <div class="father">
        <div class="son"></div>
        <div class="son"></div>
        <div class="son"></div>
    </div>
</body>

</html>

BFC的作用

作用1：可以取消盒子margin塌陷

如下所示，给子盒子设置一个margin-top值，父盒子会因为子盒子的margin-top而与上面元素产生一段margin-top值的距离，这就称为margin塌陷，同时子盒子不会产生margin-top的值在父盒子内。

解决办法：产生BFC，设置overflow：hidden

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>BFC</title>
    <style>
        .father{
            width: 200px;
            height: 300px;
            background-color: crimson;
            /* overflow: hidden; */
        }
        .son{
            width: 100px;
            height: 100px;
            background-color: darkblue;
            margin-top: 40px;
        }
    </style>
</head>

<body>
    <div class="father">
        <div class="son"></div>
    </div>
</body>

</html>

作用2：阻止浮动元素被覆盖

如果子盒子最后一个盒子因为没设置浮动，所以会被上面浮动的子盒子覆盖。

解决办法：让没有浮动的盒子产生BFC，即对其设置overflow：hidden，他就会自身成为一个独立的区域，与区域外部无关

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>BFC</title>
    <style>
        .son{
            width: 200px;
            height: 200px;
            background-color: darkorchid;
            float: left;
        }
        .son-last{
            width: 200px;
            height: 300px;
            background-color: darkslategrey;
             /* overflow: hidden; */
        }
    </style>
</head>

<body>
    <div class="father">
        <div class="son"></div>
        <div class="son"></div>
        <div class="son-last"></div>
    </div>
</body>

</html>

2.link与@import

1.link是HTML标签，@import是css提供的。
2.link引入的样式页面加载时同时加载，@import引入的样式需等页面加载完成后再加载。
3.link没有兼容性问题，@import不兼容ie5以下。
4.link可以通过js操作DOM动态引入样式表改变样式，而@import不可以

3.圣杯布局和双飞翼布局

圣杯布局

header和footer各自占领屏幕所有宽度，高度固定。

中间的main是一个三栏布局。

三栏布局两侧宽度固定不变，中间部分自动填充整个区域。

中间部分的高度是三栏中最高的区域的高度。

1.浮动实现
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>圣杯布局-浮动实现</title>
    <style>
        * {
            margin: 0;
            padding: 0;
        }

        header,
        footer {
            height: 100px;
            background: #666;
            text-align: center;
            line-height: 100px;
        }

        main {
            padding: 0 300px;
            height: 500px;
            line-height: 500px;
            text-align: center;
        }

        #middle {
            float: left;
            width: 100%;
            background: blue;
        }

        #left {
            float: left;
            width: 300px;
            margin-left: -100%;
            background: #0c9;
            position: relative;
            left: -300px;
        }

        #right {
            float: left;
            width: 300px;
            margin-left: -300px;
            background: #0c9;
            position: relative;
            right: -300px;
        }
    </style>
</head>

<body>
    <header>header</header>
    <main>
        <div id="middle">middle</div>
        <div id="left">left</div>
        <div id="right">right</div>
    </main>
    <footer>footer</footer>
</body>

</html>

双飞翼布局

与圣杯布局相比，DOM结构不同，圣杯布局是main包含中间所有部分，而双飞翼布局是container只包含中间部分的中间部分

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            text-align: center;
        }

        header,
        footer {
            height: 100px;
            background: #666;
            line-height: 100px;
        }

        #container {
            float: left;
            width: 100%;
            line-height: 500px;
            height: 500px;
            background: blue;
        }

        #left {
            float: left;
            width: 300px;
            height: 500px;
            line-height: 500px;
            margin-left: -100%;
            background: #0c9;
        }

        #right {
            float: left;
            width: 300px;
            height: 500px;
            line-height: 500px;
            margin-left: -300px;
            background: #0c9;
        }

        #middle {
            margin: 0 300px;
        }

        footer {
            clear: both;
        }
    </style>
</head>

<body>
    <header>header</header>
    <div id="container">
        <div id="middle">middle</div>
    </div>
    <div id="left">left</div>
    <div id="right">right</div>
    <footer>footer</footer>
</body>

</html>

6.HTML全局属性(包含H5)有哪些？

accesskey：设置快捷键来获取元素焦点

class：为元素设置类标识

contenteditable：指定元素内容是否可编辑

data-xxx：为元素增加自定义属性（属性名不包含大写字母，data-后面至少有一个字符）

dir：设置元素文本方向（默认ltr；rtl）

draggable：设置元素是否可拖拽

hidden：对元素进行隐藏

id：元素id，文档内唯一

lang：元素内容的语言

spellcheck：是否启动拼写错误检查(拼写错误的底下出现红波浪线)

style：行内css样式

tabindex：设置元素可以获得焦点，通过tab键控制次序

title：规定元素有关的额外信息，鼠标放置该元素上面显示title内容

dropzone：设置元素拖放类型（copy|move|link,H5新属性，主流均不支持）

translate：元素和子孙节点内容是否需要本地化（主流均不支持）

7.页面隐藏元素的方法有哪些

占据空间方法

opacity: 0 会被子类继承，且子类没办法再显示

visibility: hidden 会被子类继承，子类可以通过设置visibility:visible来显示 不会触发该元素已经绑定的事件 动态修改此属性会引起重绘

z-index: -99999 要配合position:absolute使用 部分重排，不影响其他图层布局

margin-left: -100%

transform: scale(0)

不占据空间方法

display: none 动态改变此属性时会引起重排

width:0,height:0,overflow:hidden

9.CSS选择器有哪些？哪些属性可以继承？

选择器： 通配符* id选择器 #xxx 类选择器 .xxx 属性选择器 标签[attribute=value] 后代选择器 > 标签选择 div

伪类：

选择器 含义
:first-child 匹配父元素的第一个子元素
:link 匹配所有未被点击的连接
:visited 匹配所有已被点击的链接
:active 匹配鼠标已经点击但还没松开鼠标的元素
:hover 匹配鼠标悬停在某元素上面的元素
:focus 匹配获得当前焦点的元素
:lang(x) 匹配lang属性等于x的元素
:nth-child(n) 匹配其父元素的第n个子元素，第一个编号记为1
:last-child 匹配父元素最后一个子元素
:nth-last-child(n) 匹配其父元素的倒数第n个子元素，第一个编号为1
:empty 匹配每个没有任何子级的元素（元素中啥也没有，包括文本也没有）

伪元素:

选择器 含义
:before 在元素之前插入生成的内容
:after 在元素之和插入生成的内容
:first-letter 匹配元素的第一个字母
:first-line 匹配元素的第一行

10.用css画一个三角形、原理？

https://www.jianshu.com/p/9a463d50e441

原理：宽高设为0，四个边框设置border-width，border-style，border-color即可，如果某个三角要变为透明，设置border-color为transparent

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>用css画一个三角形</title>
    <style>
        div {
            width: 0;
            height: 0;
        }

        #div1 {
            border-bottom: 50px solid blue;
            border-left: 50px solid green;
            border-right: 50px solid yellow;
            border-top: 50px solid red;
        }

        #div2 {
            border-bottom: 50px solid blue;
            border-left: 50px solid transparent;
            border-right: 50px solid transparent;
            border-top: 50px solid transparent;
        }

        #div3 {
            position: relative;
            border-bottom: 60px solid black;
            border-left: 60px solid black;
            border-right: 60px solid transparent;
            border-top: 60px solid transparent;
        }
        /* 利用伪元素实现带边框的三角形 */
        #div3::after {
            content: '';
            position: absolute;
            left: -54px;
            top: -46px;
            border-bottom: 50px solid blue;
            border-left: 50px solid blue;
            border-right: 50px solid transparent;
            border-top: 50px solid transparent;
        }

        #div4 {
            position: relative;
            border-bottom: 100px solid blue;
            border-left: 50px solid transparent;
            border-right: 50px solid transparent;
        }

        /* 利用伪元素实现三角形空心箭头 */
        #div4::after {
            content: '';
            left: -40px;
            top: 20px;
            position: absolute;
            border-bottom: 80px solid #fff;
            border-left: 40px solid transparent;
            border-right: 40px solid transparent;
        }
    </style>
</head>

<body>
    <div id="div1"></div>
    <div id="div2"></div>
    <div id="div3"></div>
    <div id="div4"></div>
</body>

</html>
